# Senior Developer Topics Index

This document lists all topics in the recommended learning order for Senior Developer preparation across Microservices, System Design, Java, REST API, LLD, DBMS, and NoSQL.

2. ‚è≥ **Scalability Patterns** - Horizontal vs vertical scaling, load balancing strategies, and scaling techniques

5. ‚è≥ **Message Queues** - Message queue patterns, pub/sub, and event streaming
6. ‚è≥ **Distributed Systems** - CAP theorem, consistency models, and distributed system challenges
